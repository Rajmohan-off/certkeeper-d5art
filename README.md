# D5art x Divershefy Professional Certification Platform

This project is a blockchain-verified professional certification platform built with modern web technologies.

## Technology Stack

- **Framework**: Vite + React
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: Redux Toolkit
- **Certificates**: S3 Storage + IPFS + Blockchain Minting

## Development Setup

1. **Install dependencies**: `npm install`
2. **Setup environment variables**: Create a `.env` file based on the provided configuration.
3. **Start development server**: `npm run dev`

## Deployment

Build the production bundle using `npm run build` and deploy the `dist` folder to your preferred hosting provider.
